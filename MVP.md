# MVP需求文档 · dewatermark.ai

> 版本：v0.1 | 日期：2026-03-19

---

## 目标

上线一个能跑通"上传图片 → 去水印 → 下载"完整流程的工具站，验证用户需求和付费意愿。

---

## 用户故事

1. 作为访客，我可以不注册直接上传图片去水印（每天3张）
2. 作为注册用户，我每天可以处理5张图片
3. 作为Pro用户，我可以无限处理图片，并获得原始分辨率输出
4. 作为任何用户，我的图片不会被服务器存储

---

## 功能清单

### F1 · 图片上传
- 支持拖拽上传 / 点击选择
- 支持格式：JPG / PNG / WEBP
- 单张大小限制：10MB
- 上传后显示预览

### F2 · 水印区域选择
- 用户在图片上手动框选水印区域（矩形框）
- 支持多框选（多个水印）
- 可撤销/重置选框

### F3 · AI去水印处理
- 调用 Clipdrop API（cleanup接口）
- 全程内存处理，不落盘存储
- 处理中显示 loading 状态
- 处理失败给出明确错误提示

### F4 · 结果预览与下载
- 左右滑动对比原图 / 处理后效果
- 一键下载处理后图片
- 免费用户输出压缩版（长边1080px）
- Pro用户输出原始分辨率

### F5 · 用量限制
- 未注册：每天3张（IP限制）
- 注册用户：每天5张
- Pro用户：无限
- 达到限额时展示升级引导

### F6 · 用户系统
- 注册/登录：邮箱+密码
- Google OAuth 一键登录
- 用量统计展示（今日已用/剩余）

### F7 · 付费订阅
- Pro 月付：$12/月
- 接入 Stripe Checkout
- 支付成功后立即解锁Pro权限
- 支持取消订阅

## 技术方案

| 模块 | 方案 |
|------|------|
| 前端 | Next.js 14（App Router）|
| 部署 | Cloudflare Pages |
| 后端API | Cloudflare Workers |
| AI去水印 | Clipdrop API（cleanup）|
| 用户认证 | NextAuth.js |
| 数据库 | Cloudflare D1（用户/用量数据）|
| 支付 | Stripe |
| 图片存储 | 不存储，内存处理后直接返回 |

## 页面结构

```
/                   首页（上传入口 + 功能介绍）
/app                工具主页面（上传/处理/下载）
/login              登录/注册
/pricing            定价页
/dashboard          用户中心（用量/订阅管理）
```

---

## 非功能需求

- 处理响应时间：< 10秒（Clipdrop API决定）
- 隐私：图片不落盘，处理完立即销毁
- SEO：首页针对核心关键词优化
- 移动端：响应式布局，可用即可（不做专项优化）

---

## 不在MVP范围内

- 批量处理
- 自动识别水印位置
- 移动端App
- 年付套餐
- 多语言
- 团队/企业版
