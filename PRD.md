## 一、产品概述

- **产品名称：** dewatermark.ai
- **产品形态：** Web工具站（海外市场）
- **核心功能：** AI智能去除图片水印，支持角标logo和半透明文字水印
- **目标：** 做成有稳定付费转化的工具站

---

## 二、目标用户

| 用户群 | 核心诉求 |
|--------|---------|
| 自媒体/内容创作者 | 素材去水印后发布 |
| 电商运营 | 竞品图参考、素材复用 |
| PPT制作者 | 清理平台水印 |
| 设计师 | 处理客户带水印素材 |

---

## 三、核心功能

### 3.1 去水印（核心）
- 支持水印类型：固定角标logo、半透明文字水印
- 处理质量：AI修复，无痕填充
- 输入格式：JPG / PNG / WEBP
- 输出格式：与原图一致，支持下载

### 3.2 处理模式
- **单张处理：** 上传 → 预览对比 → 下载
- **批量处理：** 多张上传 → 队列处理 → 打包下载（付费）

### 3.3 免费/付费分层

| 功能 | 免费 | Pro付费 |
|------|------|---------|
| 单张处理 | ✅ 每天5张 | ✅ 无限 |
| 批量处理 | ❌ | ✅ |
| 输出分辨率 | 压缩版（1080px） | 原始分辨率 |
| 处理速度 | 普通队列 | 优先队列 |

### 3.4 用户系统
- 注册/登录：邮箱 + Google OAuth
- 用量统计：今日剩余次数
- 付费订阅管理

---

## 四、付费模式

- **免费：** 每天5张，压缩分辨率
- **Pro 月付：** $12/月
- **Pro 年付：** $99/年
- **支付：** Stripe

---

## 五、技术方案

| 模块 | 方案 |
|------|------|
| 前端 | Next.js（Cloudflare Pages托管）|
| 后端 | Cloudflare Workers（无服务器）|
| AI模型 | Clipdrop API（cleanup接口）|
| 支付 | Stripe |
| 认证 | NextAuth.js |
| 数据库 | Cloudflare D1（SQLite，用量统计/用户数据）|
| 存储 | **不存储图片** — 内存处理，处理完直接返回，隐私友好 |

## 六、MVP范围

**做：**

4. Stripe Pro月付
